O que são listas?

Listas são conjuntos estruturais de dados organizados de forma linear dentro da mesma, podendo ser acessados diretamente utilizando um índice, cujo número representa a sua posição dentro daquela lista em que a mesma faz parte.

Como criar uma lista?

Para criar uma lista, utilizamos os símbolos de "colchetes", sendo eles "[" e "]".
Dentro do código, seria algo similar a:

    
    	lista = []

Para inserir dados dentro de uma lista, preenchemos, dentro dos colchetes, cada um dos itens da lista, cada um deles dentro de aspas simples (') e separados por vírgula.
Dentro do Python, seria algo similar a:

    
    	lista = []
    	lista = ['tigre', 'onça', 'leão', 123, 456]

A pergunta que fica agora é: como podemos, de alguma forma, ler estes dados presentes dentro de uma lista?

Antes de tudo, deve-se ficar claro que em uma lista, cada um dos itens presentes na mesma possui um índice, servindo para identificá-los. Na teoria, seria como no código abaixo:

    
    	lista = []
    	lista = ['tigre', 'onça', 'leão', 123, 456]
    		    0        1       2     3    4

Desta forma, a palavra "tigre" irá receber, teoricamente, o Índice de número 0.
A palavra "onça" recebe o índice de número 1.
A palavra "leão" recebe 2, e assim por diante, até o último item da lista.

Caso queiramos saber qual o item de uma lista nós inserimos, especificamos qual lista queremos identificar um valor, e dentro dos colchetes, colocamos o índice desejado.
Tendo como referência a lista acima, caso queiramos saber qual valor tem o índice de número 2, no código, ficaria da seguinte forma: 

    
    	lista[2]

Neste caso, o valor retornado seria equivalente a "leão", pois o mesmo possui o índice de número 2 dentro da nossa lista.

Através da função len(), podemos identificar também qual o tamanho da lista que inserirmos dentro dos parênteses.
No código, ficaria desta forma:

    
    	len(lista)

Métodos das Listas

Aqui, veremos métodos possíveis de realizarmos com as listas que criamos. Com elas, conseguimos incluir ou remover elementos de uma lista, por exemplo.

Temos diversos tipos de métodos, mostrados na tabela abaixo:

NomeDoMetodo Funcionalidades
append() Insere item no final da lista
insert() Insere na tabela de acordo
com o índice
pop() Remove último item da lista
remove(item) Remove o item do índice
indicado
sort() Ordena os itens da lista
reverse() Inverte as posições da lista
count() Retorna o número de vezes que
o item aparece na lista


Para conseguirmos utilizar um método com uma lista, faremos da seguinte forma:

    
    	lista.nomeDoMetodo()
