Neste tópico sera explicado as variáveis utilizadas no Python.

Para Iniciar vamos ver como declarar uma variavel:

Para se declarar uma variável basta inicia-la com uma letra ou underline. O python nao reconhece variaveis que começam com simbolos ou numeros.
Porém após o primeiro caracter pode ser utilizado numeros ou simbolos.

Lembre-se que o Python é sensivel a detecção entre maiúsculas e minúsculas.

Agora vamos ver quais são as variaveis mais utilizadas:

Primeiramente, o Python não lhe obriga a especificar qual o tipo da variável que você ira usar. Agora vamos aos tipos de variáveis:

  • Int
  • Float
  • String
  • Bool
  • Complex

O comando type é utilizado para verificar o tipo da variável.

Explicando cada variável:

-INT: Do tipo inteiro.

-FLOAT: Possui um ponto entre as casas decimais, facilitando para calculos matematicos.

-STRING: Variavel de texto, utilizado de nomes, a pequena frases e ate grandes textos.

-BOOL: Variavel primitiva que é utilizada por ter apenas dois valores, conhecida por ser utilizada com True/ False ou 0/1.

-COMPLEX: Representado por um int (Ou float) somado a um numero imaginario (sendo o sufixo do numero imaginario um J)

Dá para perceber o motivo que a linguagem vem crescendo tanto nos ultimos anos, justamente por ser extremamente facil de compreende-lo.
