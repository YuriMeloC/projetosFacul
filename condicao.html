As estruturas condicionais estão presentes em quase todas das principais linguagens de programação existentes no mercado.

Em Python, devido sua facilidade de desenvolvimento dos códigos, a implementação das condições if, else e elif seguem essa mesma linha.

O significado para cada uma dessas estruturas de condição é de que, avaliando uma determinada expressão e de acordo com seu resultado, conseguimos executar uma determinada ação.

Esses três tipos de condições podem ser classificadas como Estruturas condicionais Simples, composta e aninhada.

Estrutura condicional Simples

Para a estrutura condicional do tipo simples, utilizamos o comando if.

Um exemplo de uso do if se aplica abaixo:

    
      idade: 18
      if idade < 10:
        print('Você não possui acesso!')

Neste caso, antes de tudo, acabamos por especificar o valor da variável idade em 18.
Em seguida, inserimos o comando if, e nela estabelecemos a seguinte condição:

SE o valor da variável idade for MENOR que o número 10
Então o sistema executa a ação representada na terceira linha, no caso, exibindo a frase "Você não possui acesso!" em tela.

Uma das facilitações da linguagem de programação em Python é poder inserir, na Estrutura Condicional Simples, diversos if dentro de da mesma estrutura.
O resultado é o seguinte:

    
      idade: 18
      if idade < 18:
        print('Você não possui acesso!')
      if idade > 18:
        print('Você possui acesso!')

No caso desta estrutura, estabelecemos que SE o valor da variável idade for MENOR do que o número 10, ele executa uma ação, exibindo a frase "Você não possui acesso!" em tela.
Se o valor da variável idade não se aplicar a primeira condição, a segunda condição é executada.
No caso, SE o valor da variável idade for MAIOR do que o número 18 a frase "Você possui acesso!" é exibida em tela.

Estrutura condicional composta

Para a estrutura condicional do tipo composta, utilizamos o comando if, seguido do comando else.
O comando if executa uma ação quando a condição for verdadeira e, se a mesma for falsa, o comando else executa uma outra ação.

Um exemplo do uso da estrutura composta, utilizando o if-else se aplica abaixo:

    
      valor = 1
      if valor < 0:
        print ('O valor é menor que zero')
      else:
        print ('O valor é maior que zero')

Neste caso, o comando if verifica o valor da variável "valor" e, SE a condição for verdadeira, ele executa a ação na linha abaixo, exibindo a frase "O valor é menor que zero" em tela.
Caso a condição for falsa, o comando else executa a linha abaixo, exibindo a frase "O valor é maior que zero" em tela.

Estrutura condicional aninhada

Ao utilizarmos as Estruturas condicionais aninhadas, utilizamos um tipo diferente do if e do else, inserindo entre os dois comandos o comando elif

O elif funciona de uma maneira semelhante ao if. Sua diferença é que o mesmo pode ser usado diversas vezes dentro de uma estrutura condicional, executando o comando seguinte se sua ação for verdadeira e senão, passando para o próximo elif sequencialmente até chegar ao comando else.

Um exemplo de como utilizar o elif está sendo exemplificado abaixo:

    
      if soma > 0:
       print ('Maior que Zero.')
      elif soma == 0:
           print ('Igual a Zero.')
      else:
       print ('Menor que Zero.')

Neste caso, o if verifica o valor de soma e, se a mesma for MAIOR que 0, a frase "Maior que Zero." é exibida em tela.
Em seguida, no elif é verificado se o valor de soma é IGUAL a 0 e se verdadeiro, a frase "Igual a Zero." é exibida em tela.
Se nenhuma dessas condições for verdadeira, o comando else é executado, exibindo a frase "Menor que Zero." em tela.
